# libmotsya-allmuisc
A multimedia library made with guncobal

## Features
- Lightweight dynamic library (`libcobolmotsys.so`)
- Compatible with all Linux distributions (Termux, Ubuntu, Fedora, etc.)
- No external dependencies (uses standard C/POSIX and GnuCOBOL APIs)
- Easy integration with COBOL/C programs via header file

## Prerequisites
- GnuCOBOL (>= 3.1.2)
- GCC (>= 8.0)
- Make (>= 4.0)
- Root privileges (for installing to `/usr/lib` and `/usr/include`)

## Build & Install
```bash
# Clone repository
git clone https://github.com/points14514/libmotsys-allmuisz.git
cd libmotsys-allmuisz

# Switch to root (required for system-wide installation)
sudo -i  # Or `su` on Termux

# Execute installation script
./setup-liballmuisz
```
